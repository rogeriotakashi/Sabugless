Disciplina: Programação Orientada a Objetos 2

O Projeto Demosoft tem como objetivo exibir informações relevantes relacionadas a diversos dados demográficos. Esses dados podem ser utilizados em diversas ocasiões, desde uma pesquisa escolar ou até mesmo para conhecimento próprio. Portanto, o Demosoft permite que uma série de dados relevantes sejam reunidos em um mesmo lugar e que os mesmos sejam disponíveis de forma prática e interativa para os usuários.

Disciplina: Estrutura de Arquivos

Projeto em Java para a disciplina de Estrutura de Arquivos na Unicamp. Este software lê um arquivos de dados, criando o índice primário, índice secundário e arquivos de lista invertida. Também, ele permite que o usuário consiga realizar buscas nestes arquivos.
